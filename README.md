# nanogenmo-2025

This is history of edits to the Wikipedia article [Theseus](https://en.wikipedia.org/wiki/Theseus)
formatted as a conversation between a large number of people. The actual commentary in the edit
summaries is ignored: the script only presents the changes that users made to the article text, as if
they had described them out loud.
